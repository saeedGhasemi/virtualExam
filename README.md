# Virtual Exam

پروژه اولیه برای سامانه برگزاری امتحانات مجازی با `Django`، `PostgreSQL` و `Tailwind CSS`.

## امکانات اولیه
- صفحه اصلی معرفی سامانه
- ساختار اپ `core`
- تنظیمات اتصال به PostgreSQL با فایل `.env`
- استایل اولیه با Tailwind

## راه‌اندازی سریع
1. ساخت فایل `.env` بر اساس `.env.example`
2. نصب وابستگی‌های پایتون و نود
3. اجرای migrate
4. اجرای build فایل CSS
5. اجرای سرور جنگو

## مدیریت کلیدهای API و امنیت
کلیدهای API (مثل کلیدهای OpenAI) را هرگز مستقیم در مخزن ذخیره نکنید. اگر کلیدی تصادفاً فاش شده (مانند کلیدی که در این گفتگو ارسال شد)، فوراً آن را در پنل ارائه‌دهنده ری‌ووک یا حذف و یک کلید جدید ایجاد کنید.

- فایل نمونه `.env.example` شامل متغیر `OPENAI_API_KEY` است؛ یک فایل `.env` بسازید و مقدار کلید را در آن قرار دهید (این فایل در `.gitignore` آمده و نباید به گیت اضافه شود).
- کلیدها را در محیط‌های Production یا CI از طریق Secrets امن (مثلاً GitHub Actions Secrets) تنظیم کنید.

اگر خواستید، می‌توانم راهنمای گام‌به‌گام برای ری‌ووک کلید یا پاک‌سازی تاریخ گیت آماده کنم.
